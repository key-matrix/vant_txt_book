# Flutter 初学者ガイド

## 1. Flutter 概要

Flutterは、Googleによって開発されたオープンソースのUIソフトウェア開発キットです。
単一のコードベースから、iOS、Android、Web、Desktop、および組み込み機器向けのアプリケーションを開発することが可能です。

---

## 2. Pythonエンジニアとしての優位性

既に Python を学習されている方は、Flutter（Dart言語）の習得において非常に有利です。
以下の比較表は、それぞれの特徴をまとめたものです。

| 項目         | Python                         | Flutter (Dart)                           |
| :----------- | :----------------------------- | :--------------------------------------- |
| **主な用途** | データ分析、AI、サーバーサイド | **マルチプラットフォームアプリ開発**     |
| **文法特性** | 簡潔で可読性が高い             | Pythonに近い平易さと静的型付けの両立     |
| **開発体験** | REPLによる即時確認             | **ホットリロード**による高速なUI反復開発 |

---

## 3. 宣言的UIの基本概念

Flutterでは「ウィジェット（Widget）」と呼ばれるコンポーネントを組み合わせてUIを構築します。
「どのような状態であるべきか」を宣言することで、フレームワークが適切に描画を管理します（宣言的UI）。

```dart
// 基本的なボタンの実装例
ElevatedButton(
  onPressed: () => print('Button pressed'),
  child: Text('Submit'),
)
```

このアプローチにより、複雑なUIも論理的かつ効率的に構築することが可能になります。

## 4. 学習ロードマップ

以下のようなステップで学習を進めることを推奨します。

1. **環境構築（1日）**: `DEV_ENV_README.md` を参照し、開発環境を整備します。
2. **UI基礎（1〜2週間）**: 基本的なウィジェットの配置とレイアウト手法を習得します。
3. **実践演習（随時）**: `EXERCISES.md` の課題に取り組みながら、学んだ知識を定着させます。
4. **最終課題の実装（以降）**: 本格的なチャットアプリの開発プロジェクトに取り組みます。
