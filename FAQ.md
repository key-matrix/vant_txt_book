# よくある質問 (FAQ)

## 1. 導入および基本概念に関する質問

**Q: プログラミング未経験でも学習は可能ですか？**
A: はい、可能です。Python等の他言語の基礎知識があれば習熟はさらに早まりますが、一歩ずつ進めることで確実にスキルを習得できます。

**Q: 開発に必要なコストについて**
A: Flutter SDKおよび主要なIDE（VS Code, Android Studio, Cursor等）は全てオープンソースまたは無料で提供されており、学習にあたってのツール費用はかかりません。
※バックエンドにクラウドサービスを使用する場合は別途費用が発生します

---

## 2. 環境構築およびツールに関する質問

**Q: Windows環境でiOSアプリの開発は可能ですか？**
A: iOSアプリのビルドおよびシミュレータの実行には macOS が必要です。ただし、基本的なFlutterの学習や Android アプリの構築は Windows 環境でも問題なく行えます。

**Q: Flutterのバージョン管理（FVM）は必要か？**
A: 複数のプロジェクトを並行して扱う場合、それぞれで要求されるFlutterのバージョンが異なることがあります。FVMを使用することで、プロジェクトごとに適切なバージョンを確実に指定でき、予期せぬビルドエラーを防ぐことができます。

**Q: 推奨される IDE は何ですか？**
A: 本教材では **AndroidStudio** または **VS Code** の使用を推奨しますが、使い慣れたエディタであれば問題ありません。

---

## 3. 実装およびデバッグに関する質問

**Q: ホットリロードが反映されない場合の対処法を教えてください。**
A: コードの保存時に自動適用されない場合は、IDE上のホットリロードボタンを押下してください。また、大幅な構造変更を行った（例：initState の変更）場合は、アプリの再起動（Hot Restart）が必要になることがあります。

**Q: エラーが発生した際の調査手順を教えてください。**
A: まずデバッグコンソールのログを確認してください。多くのメッセージの中から、「Exception」や「Error」と記述された箇所を特定し、その内容を元に調査を行います。公式ドキュメントや、修正後のコードをAIにレビューしてもらう手法も有効です。

---

## 4. 学習の進め方に関する質問

**Q: 効率的な学習の順序はありますか？**
A: まずはじめに読むガイド（README.md）を通読し、次に環境構築ガイド（DEV_ENV_README.md）を完了させてください。その後、ウィジェットの基礎を固めながら、小規模なアプリ（TODOリスト等）の構築へと段階的に進むのが確実です。
