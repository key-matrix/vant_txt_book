| ステップ | 対象ドキュメント | 学習内容（目的） |
| :---: | :--- | :--- |
| **1** | [README.md](./README.md) | 概要と基本概念の理解 |
| **2** | [DEV_ENV_README.md](./DEV_ENV_README.md) | 開発に必要な環境構築 |
| **3** | [WIDGET_README.md](./WIDGET_README.md) | UIコンポーネント（部品）の学習 |
| **4** | [EXERCISES.md](./EXERCISES.md) | 学んだ知識を活かす実践演習課題 |
| **5** | [FINAL_PROJECT.md](./FINAL_PROJECT.md) | 総集編となる最終プロジェクトの開発 |

## 3. 学習上のアドバイス

- **段階的な学習**: 基礎を理解した上で次のステップへ進むことが、結果として最短の学習経路となります。
- **実践の重視**: 本ドキュメントのサンプルコードを実際に自分の環境で入力し、動作を確認することを推奨します。
- **リソースの活用**: 詰まった際にはまずFAQを確認し、解決しない場合は公式ドキュメントやコミュニティも参照してください。

まずは、[はじめに読むガイド](./README.md) から開始してください。
